Is Microsoft finally doing things right?

Creator: Brayden Peacock | Date: 4/26/2026

Microsoft has released Windows 11 build 26300.2289 on 4/26/26. Let's talk about it.

1st Change: Microsoft has added the new spinny animation from Windows 10x and 11 into even more places.

2nd Change: Microsoft has said that the taskbar will be more flexible now being able to go to the top, left, and right. About time!

3rd Change: Microsoft said they would let users skip Windows Update in OOBE.

4th Change: Microsoft has updated Run. To get the new run, go to Settings > System > Advanced and enable 'Modern Run'.

5th and Final: In Windows Update, Dev and Canary were merged into the Experimental Channel.

Thanks for reading this blog and have a great day.
